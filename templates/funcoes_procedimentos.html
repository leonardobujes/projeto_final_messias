Funções e Procedimentos

Conceito

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas podem receber dados de entrada (argumentos/parâmetros) e podem retornar um valor. Em Python, não há uma distinção sintática estrita entre "funções" (que retornam valor) e "procedimentos" (que não retornam valor explícito, retornando None por padrão). Ambos são definidos com a palavra-chave def.

Aplicação

São fundamentais para organizar o código, torná-lo mais modular, legível e reutilizável. Evitam a repetição de código e facilitam a manutenção e o teste de programas.

Exemplo de Código


# Função que retorna um valor (tradicionalmente "função")
def somar(a, b):
    """Esta função retorna a soma de dois números."""
    return a + b

# Função que não retorna um valor explícito (tradicionalmente "procedimento")
def saudar(nome):
    """Esta função imprime uma saudação."""
    print(f"Olá, {nome}!")

# Usando as funções
resultado_soma = somar(5, 3)
print(f"O resultado da soma é: {resultado_soma}")

saudar("Mundo")

# Saída esperada:
# O resultado da soma é: 8
# Olá, Mundo!
