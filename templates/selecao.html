Estruturas de Seleção

Conceito

As estruturas de seleção (ou condicionais) permitem que um programa execute diferentes blocos de código dependendo se uma determinada condição é verdadeira ou falsa. As principais em Python são if, elif (else if) e else.

Aplicação

São usadas para tomar decisões no código. Por exemplo, verificar se um usuário tem permissão para acessar um recurso, determinar se um número é positivo ou negativo, ou reagir a diferentes entradas do usuário.

Exemplo de Código


idade = 18
permissao_pais = False

if idade >= 18:
    print("Pode entrar na festa.")
elif idade < 18 and permissao_pais:
    print("Pode entrar na festa com permissão dos pais.")
else:
    print("Não pode entrar na festa.")

# Saída esperada: Pode entrar na festa.
