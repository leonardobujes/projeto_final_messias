Estruturas de Repetição

Conceito

Estruturas de repetição (ou laços) permitem executar um bloco de código múltiplas vezes. Python oferece principalmente os laços for (para iterar sobre sequências) e while (para repetir enquanto uma condição for verdadeira).

Aplicação

Utilizadas para processar itens em uma lista, ler linhas de um arquivo, repetir uma ação até que uma condição seja satisfeita, ou realizar cálculos que exigem múltiplas iterações.

Exemplo de Código


# Exemplo com for
print("Contagem com for:")
for i in range(1, 6):  # De 1 a 5
    print(i)

# Exemplo com while
print("\nContagem com while:")
contador = 1
while contador <= 5:
    print(contador)
    contador += 1

# Saída esperada:
# Contagem com for:
# 1
# 2
# 3
# 4
# 5
#
# Contagem com while:
# 1
# 2
# 3
# 4
# 5
