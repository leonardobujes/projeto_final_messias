Vetores e Matrizes

Conceito

Em Python, "vetores" são comumente representados por listas (list), que são coleções ordenadas e mutáveis de itens. "Matrizes" (arrays bidimensionais) podem ser implementadas como listas de listas. Para operações numéricas mais avançadas, bibliotecas como NumPy são frequentemente usadas, oferecendo arrays mais eficientes.

Aplicação

Vetores são usados para armazenar coleções de dados, como uma lista de nomes, temperaturas, ou notas. Matrizes são úteis para representar dados tabulares, como planilhas, imagens (pixels), ou sistemas de equações lineares.

Exemplo de Código


# Vetor (Lista em Python)
print("Vetor (Lista):")
numeros = [10, 20, 30, 40, 50]
print(f"Primeiro elemento: {numeros[0]}")
print(f"Último elemento: {numeros[-1]}")
numeros.append(60)
print(f"Lista após adicionar 60: {numeros}")

# Matriz (Lista de Listas em Python)
print("\nMatriz (Lista de Listas):")
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(f"Elemento na linha 1, coluna 2: {matriz[0][1]}") # Linhas e colunas começam em 0
print("Imprimindo a matriz:")
for linha in matriz:
    print(linha)

# Saída esperada:
# Vetor (Lista):
# Primeiro elemento: 10
# Último elemento: 50
# Lista após adicionar 60: [10, 20, 30, 40, 50, 60]
#
# Matriz (Lista de Listas):
# Elemento na linha 1, coluna 2: 2
# Imprimindo a matriz:
# [1, 2, 3]
# [4, 5, 6]
# [7, 8, 9]
